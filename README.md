# Тестовое задание для Workmate

Данный репозиторий является выполненным тестовым заданием для Workmate.

## Стек технологий

- Python 3.12;
- pytest;
- pytest-cov;
- tabulate;
- cli;
- csv;
- argparse.

## Запуск проекта

Для запуска проекта необходимо установить используемые библиотеки:

```bash
pip install -r requirements.txt
```

После этого для запуска необходимо прописать команду:

```bash
python main.py --files [FILES] --report REPORT
```
